== Description ==

This plugin gives the facility for hiding and showing the admin menu of the side / top bar, it also gives you the options to hide the plugin lists, WordPress version, rename, and style the brand of WordPress into your brand. 

Ever want to style the appearance of the WordPress admin pages by hiding the menu, renaming the brand, changing the logo, etc.

== Which Reason Makes the WordPress WP Brand Plugin Outstanding? ==
There are the following reasons that make the WordPress WP Brand Plugin Outstanding.

1. HIDE ADMIN MENU AND ADMIN BAR
You can hide the items in the admin menu and admin bar.

2. HIDE ADMIN MENU ITEMS FOR SPECIFIC USERS
If you have multiple users on your website, you will sometimes need certain people to have specific permissions. It is the best way to do it for your particular user role.

3. Hide Plugin
You can hide all the plugins or particular plugins from the plugin page.

4. HIDE WORDPRESS VERSION
You can easily hide WordPress version by one click.

5. REBRAND (NAME, LOGO, WELCOME MESSAGE)
You can easily rename the WordPress into your brand name.

== Frequently Asked Questions ==

= How can I show hide menus? =

You can show menu that is hide by this plugin doing the uncheck those menus from the menu list in form.

= What should you do when you hide all the menu by this plugin? =

You should open below URL
YOUR-WEBSITE-URL/wp-admin/options-general.php?page=mwb

Update necessary setting to show any menu again by this URL.
